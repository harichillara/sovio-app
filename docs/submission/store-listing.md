## iOS export compliance (Encryption Info)

- Uses encryption: **Yes**
- Qualifies for exemption: **Yes** — "Uses only encryption that is (a) standard
  encryption built into iOS/iPadOS" and (b) HTTPS/TLS for network traffic.
- CCATS / ERN required: **No**
- Add `ITSAppUsesNonExemptEncryption = false` to the final shipping
  `Info.plist` (or via EAS `ios.config.usesNonExemptEncryption = false`) so
  App Store Connect stops prompting on every upload.
